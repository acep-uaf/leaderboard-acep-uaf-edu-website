# leaderboard-acep-uaf-edu-website
ACEP Red Vs Blue Leaderboard Website

# Installation

```
git clone https://github.com/acep-uaf/leaderboard-acep-uaf-edu-website
```

```
cd leaderboard-acep-uaf-edu-website
```

```
npm install
```

# Running Website

```
npm start
```

Point your browser to:
- http://127.0.0.1:8081/
